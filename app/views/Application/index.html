Functional Therapeutic Chemical Classification System (FTC)

Mode of Action

The Functional Therapeutic Chemical Classification System (FTC) defines over 20'000 mechanisms and modes of action for approuved drugs. The resource abstracts away from the traditional chemical structure-based approach and focuses solely on the mode of action of drugs. The classification sorts therapeutic agents based on their function in the human body.

Drug Repurposing

The classification helps to understand the polypharmacology of drugs. Modes of actions categories are the starting points to find new diseases indications. The Functional Therapeutic Chemical Classification System (FTC) helps to generate drug repurposing hypotheses to be tested in the laboratory.

Open Source

From the web application code to drug repositionning hypotheses, everything is open and free. The Functional Therapeutic Chemical Classification System (FTC) integrates and leverages the public content of the Gene Ontology, DrugBank, Uniprot and GO annotations into a unique resource.
